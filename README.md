# Advanced-Computing-Project
Machine learning-based early warning system for US stock market corrections using macroeconomic indicators, SHAP explainability, and an interactive risk dashboard.

This project develops a binary classification pipeline to predict S&P 500 market corrections (≥10% decline within 3 months) using 34 years of monthly macroeconomic data (1991–2024). Three models are trained and compared — Logistic Regression, Random Forest, and XGBoost — with SMOTE for class imbalance, threshold optimisation for crash recall, and SHAP analysis for interpretability. An interactive Plotly dashboard visualises real-time model predictions and macroeconomic indicator trends.
